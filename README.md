# osmcha-charter-project
Discussion around OSMCha + OSMUS

> [!NOTE]
> OSMCha operated as a Charter Project of OpenStreetMap US from 2023 through the end of 2025. In January 2026, the project transitioned to become a Program of OSM US. This repository is archived for historical reference but will not be updated.
>
> More info here: https://openstreetmap.us/news/2026/01/maproulette-and-osmcha-become-programs/
